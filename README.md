# 介绍 Rokid Homebase Stack

Rokid 智能家居 Homebase，是一套以智能语音交互为主的智能家居接入框架。 用于实现语音指令封装，设备发现， 设备状态查询，与设备控制。

Rokid Homebase 有如下特点：

- 开放协议， 使用开放协议 HTTP， TCP， SSDP
- 基于协议， 无 SDK
