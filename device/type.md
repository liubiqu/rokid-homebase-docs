# 设备类型

目前支持的设备类型以及定义：

- `curtain`: 窗帘
- `light`: 灯
- `irController`: 红外控制器
- `socket`: 插座
- `switch`: 开关
- `panel`: 面板
- `tv`: 电视
- `fan`: 电风扇
- `remoteController`: 遥控器
- `ac`: 空调
- `airPurifier`: 空气净化器
- `scene`: 场景
- `cleanBot`: 扫地机器人
- `projector`: 投影仪
- `dehumidifier`: 除湿器
- `thermostat`: 调温器
- `humidifier`: 加湿器
- `door`: 门锁
- `floorHeater`: 地暖

> 如果上面类型无法满足需求， 需要添加新的设备类型， 请向我们 [提交 Issues](https://github.com/Rokid/rokid-homebase-docs/issues)
